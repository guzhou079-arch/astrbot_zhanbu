# 🔮 AstrBot 占卜插件

周易占卜插件，支持**梅花易数**和**六爻**两大占卜体系，占卜结果由 LLM 以 bot 人设风格进行解读。

## ✨ 功能

### 梅花易数
- **时间起卦**：根据当前年月日时自动起卦（`/梅花 [问题]`）
- **报数起卦**：用户报两个数字起卦（`/梅花 数字1 数字2 [问题]`）
- **心念起卦**：随机数起卦（`/占卜 [问题]` 或 `/算卦 [问题]`）

### 六爻占卜
- **铜钱摇卦**：模拟三枚铜钱六次投掷（`/六爻 [问题]`）

### 智能解读
- 自动调用 LLM，以 bot 当前人设风格解读卦象
- 综合分析本卦、变卦、互卦、体用关系
- 对所问之事给出具体建议

### 自然语言触发
- 支持「帮我算一卦」「来占卜一下」「给我算命」等自然语言触发

## 📌 命令列表

| 命令 | 说明 |
|------|------|
| `/占卜 [问题]` | 心念起卦（随机） |
| `/算卦 [问题]` | 同上 |
| `/梅花 [问题]` | 时间起卦 |
| `/梅花 数字1 数字2 [问题]` | 报数起卦 |
| `/六爻 [问题]` | 铜钱摇卦法 |
| `/卦象帮助` | 显示帮助信息 |

## 🛠️ 安装

在 AstrBot 管理面板中安装，或手动 clone 到 plugins 目录：

```bash
cd /path/to/astrbot/data/plugins
git clone https://github.com/guzhou079-arch/astrbot_plugin_divination.git
```

重启 AstrBot 即可使用。

## 📝 说明

- 本插件为娱乐性质，占卜结果仅供参考
- 解读会自动使用 bot 当前配置的人设风格
- 需要配置至少一个 LLM provider 才能使用 AI 解读功能
